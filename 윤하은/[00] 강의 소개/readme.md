- 기반 기술에 대한 공부
    - 스프링 핵심 원리, HTTP

### MVC 2편에서 학습할 내용

- 지금까지 학습한 내용을 기반으로 백엔드 웹 개발 활용 기술을 학습한다.
- 스프링 MVC 2편 : 백엔드 웹 개발 활용 기술
    - 타임리프
    - 메시지, 국제화
    - 검증(validation)
    - 쿠키, 세션
    - 필터, 인터셉터
    - 예외처리
    - 타입 컨버터
    - 파일 업로드
    - 활용

### 학습 목표

- 개념부터 활용까지 예제 코드로
- 방대한 실무 웹 활용 기술 완성
- 더 깊이 있는 백엔드 개발자로 성장
